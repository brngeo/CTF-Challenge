# What is CTF or Capture The Flag?
Capture the Flag (CTF) is a cybersecurity competition
Where participants find hidden text strings, \
Known as "flags," within intentionally vulnerable programs,
Websites, or network systems.

These flags typically follow a standard format, such as flag{...} or CTF{...}, \
and can be discovered through various methods including web source code inspection, \
binary exploitation, and network analysis.

Flag Examples:
Item -Flag{Here_is_the_fl@g}
Item -flag{secretpassword}
Item -CTF{this_is_a_flag_in_base64!}
Item -Flag{This_is_Br4y4n}
